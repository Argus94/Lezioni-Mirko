I primi attributi

Gli attributi, come gli aggettivi, forniscono informazioni aggiuntive al tag
Ne esistono molti, per ora ne elencheremo solo alcuni:

  • lang = Specifica il linguaggio all'interno di un paragrafo o un tag di tipo testuale
  • title = Un testo che appare quando passi sopra al tag con il cursore
  • href = Indica la destinazione del link
  • alt = Un testo che appare quando una risorsa non è disponibile.
  • style = Contiene le info di stile per un documento o parte di esso.
    Se non si usa un documento css, si può inserire il codice css qui dentro.
    Attenzione però: deve essere breve o la pagina rallenta
  • target = Specifica come il link dovrà essere aperto
    • _self = è di default, apre la pagina nella stessa finestra
    • _blank = Apre la pagina in una nuova finestra
    • _parent = Apre il documento nella finestra in cui si trova la pagina
    • _top = Si sovrappone alla pagina
  • src = Indica la risorsa da cui attingere (source), come per l'immagine da prendere ad esempio
  • width = Indica la larghezza di un elemento, che può essere un box, o un'immagine
  • height = Indica l'altezza di un elemento, \\
  • class = è indispensabile, soprattutto per il css perchè identifica una classe di elementi (o tag) e permette quindi di usare la stessa classe su più tag
  • id = Viene usato per identificare un singolo tag con appunto un id che può servive per i link, come abbiamo già visto, o per modificare il singolo tag tramite CSS
