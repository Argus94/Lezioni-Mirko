L'Iframe

L'Iframe è come una finestra ritagliata all'interno di una pagina. All'interno di essa si può vedere un'altra pagina.
Iframe è l'abbreviazione di Inline Frame.

Citando alcuni attributi:

  • L'attributo scrolling, obsoleto per l'html5, si usa il css, si decide se dare o meno le barre di scorrimento ad un iframe
  • L'attributo frameborder obsoleto per l'html5, //, permette di incidare il margine o meno di un frame, 0 era assenza di margine, 1 la sua presenza.
  • L'attributo seamless permette di rimuovere le barre di scorrimento

    • è possibile anche pubblicare un pdf tramite tag iframe.

    I metadati

    I metadati si scrivono nel tag head, nella testa, non è quindi visibile agli utenti ma svolge diversi compiti, non prevede un tag di chiusura.
    Gli attributi più utilizzati sono:

    • name: indica la proprietà
    • content: il valore di questa proprietà
