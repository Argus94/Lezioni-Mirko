Etichette per i controlli del modulo
<label> <fieldset> <legend>

Quando sono stati introdotti i moduli si metteva del semplice testo accanto ad ognuno di essi.
Con l'introduzione del tag <label> le cose sono cambiate, viene così usato per identificare ogni modulo. Rende il codice più chiaro ed è supportato dai programmi utilizzati da chi ha difficoltà visive.
Può essere usato in due modi:

  1. Per racchiudere sia la descrizione testuale che l'elemento di input del modulo
  2. Per indicare il modulo di controllo a cui fa riferimento utilizzando l'attributo for
L'attributo for, in combinazione con l'id, viene anche utilizzato per estendere l'area di selezione anche all'etichettà, per quanto riguarda radio button e checkbox.

L'elemento <optgroup>

Esso è legato al tag <select>. Permette di creare dei gruppi tra le varie scelte.

Alcuni approfondimenti sul tag <select>

Esistono altri attributi legati a questo tag che possono tornare utili.
Tra questi abbiamo:

  • multiple: Permette di selezionare più opzioni
  • required: Lo rende un elemento necessario per la compilazione del form
  • size: Definisce il numero di elementi visibili nella lista a tendina
  • form: Permette di indicare a quale form è legato.


Per farlo funzionare bisognerebbe informare l'utente che è possibile compiere più scelte tramite la combinazione dei tasti CTRL + click mouse, per questo è più user friendly utilizzare le checkbox in questi casi.
